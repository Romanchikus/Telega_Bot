Тестовое задание для разработчика Python



Основная задача бота: делать скриншот сайта исходя из ссылки, которую боту отправляет пользователь



Описание функционала бота: пользователь отправляет боту ссылку на сайт, после этого бот делает скриншот сайта и отправляет пользователю изображение.



Требования:

бот должен быть в Telegram(можно создать отдельный бот для демонстрации);
токен бота должен тянуться из config-файла;
бот должен самостоятельно сделать скриншот сайта(не использовать готовые сервисы для скриншотов);
для создания скриншота сайта можно использовать любую библиотеку;
скриншот сайта должен быть полноэкранным(1920хfull_height);


Дополнительные требования:

будет плюсом реализовать функционал создания скриншота сайта, который будет убирать все статические элементы на сайте(кнопка перехода к началу страницы, JivoSite и т.д.). Главная задача этого требования - чтобы на скриншоте не было повторяющихся элементов.
